### Uso
Configure o MongoDB  
Instale as Dependencias do Projeto:
  - npm i
  - npm i express
  - npm i mongoose
  - npm i axios  
  
Instale em modo desenvolvimento:
  - npm i --safe -dev nodemon
  
### Postman
Para testar a API, baixe e instale o Postman (https://www.getpostman.com/downloads/)
